Publications

Selected work in LiDAR data, computer vision, and large-scale annotation benchmarks.

ISPRS 2023
Cairo, Egypt
YUTO: A Large Scale Aerial LiDAR Data Set for Semantic Segmentation

Introduced a large-scale aerial LiDAR benchmark dataset for semantic segmentation of urban forest scenes, enabling standardized evaluation across deep learning methods and addressing the lack of publicly available annotated point cloud datasets at scale.

ICPR 2022
Montreal, Canada
YUTO Tree-5000: A Large-scale Airborne LiDAR Data for Single Tree Detection

Presented YUTO Tree-5000, a large-scale annotated airborne LiDAR dataset of 5,000+ individually labeled tree instances, designed to establish a reproducible benchmark for single-tree detection algorithms and improve evaluation consistency across the field.

Master Thesis

Deep convolutional neural network based single tree detection using volumetric module from airborne LiDAR data

Developed a volumetric deep CNN architecture for detecting individual trees from airborne LiDAR point clouds. Achieved state-of-the-art performance on self-benchmarked annotation data and was awarded Best Master's Thesis at York University (2022).
